# EnProject
줄거리와 등장인물만 입력하여 소설과 이미지를 창조하는 사이트입니다.
# 기술스택
- Python
- Django
- HTML5
- CSS
- Bootstrap5

# 기능
- 등장인물,줄거리 입력시 소설과 이미지 창조 구현
- 만들어진 소설들과 이미지 게시판 구현
- 로그인,회원가입 구현
- 내가만든 소설들 확인 페이지 구현
